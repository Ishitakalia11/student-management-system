# Student Management System

## 📌 Project Description
A simple Student Management System using MySQL that manages student records with basic CRUD operations.

## 🛠️ Technologies Used
- MySQL
- SQL Workbench

## 📂 Features
- Create database and table
- Insert student records
- View student data

## 📊 Table Structure
- id (Auto Increment Primary Key)
- name
- roll_no
- course
- marks

## 🚀 How to Use
1. Open MySQL Workbench
2. Run the SQL script (student_db.sql)
3. Check data using SELECT query
